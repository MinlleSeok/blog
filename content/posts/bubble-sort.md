# Bubble Sort

- 옆에 있는 값과 비교해서 더 작은 값을 앞으로 보내면 어떨까?

## c

```c
#include <stdio.h>

int bubble_sort(void) {
	// i, j = for
	// temp = swap
	int i, j, temp;

	int array[10] = {1, 10, 5, 8, 7, 6, 4, 3, 2, 9};
	for (i = 0; i < 10; i++) {
		for (j = 0; j < 9 - i; j++) {
			if(array[j] > array[j + 1]) {
				temp = array[j];
				array[j] = array[j + 1];
				array[j + 1] = temp;
			}
		}
	}
	for(i = 0; i < 10; i++) {
		printf("%d ", array[i]);
	}
	return 0;
}

int main(void) {
	bubble_sort();	
}
```

## 시간 복잡도

- 10 + 9 + 8 + ... + 1
- 등차수열

- N x (N + 1) / 2
- O(N x N)

- 버블 정렬의 경우,
- 자리를 바꾸는 연산(스왑)을 계속 수행(매번)하기 때문에
- 실제 수행 시간이 더 느린 정렬입니다.
