## 벡터 Vector

- 크기 magnitud
- 방향 direction

속도(speed) - 5mph (크기) : 스칼라(scalar)  
속력(velocity) - 동쪽 (east) 5mph (크기) : 벡터(vector)

```t
v = (5, 0) = [5]
             [0]
길이 : 5
```

```t
v = (3, 4) = [3]
             [4]
길이 : 5 (피타고라스 3^2 + 4^2 = 5^2)
```

## 튜플 Tuple

```t
|R  
R^2
```

### 실수좌표공간 Real Coordinate Space

- 실수값을 가지는 모든 2-튜플
- 튜플 : 순서가 정해진 리스트

#### R^3 : 3D real Coordinate space

```t
x = [0 0 0]  
b = [-1 5 2]  
c = [i(허수) 0 1]
```

- R^n : n - dimensional real coordinate space
