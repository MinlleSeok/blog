# 웹사이트 및 브라우저 주요 공격 유형

1. 크로스사이트 요청 위조
2. 크로스사이트 스크립팅
3. SQL 인젝션
4. 비트플리핑 공격

## 크로스사이트 요청 위조

- 새로운 공격은 아니지만, 간단하고 매우 위협적인 공격입니다.
- 로그인한 피해자의 브라우저가 취약한 웹 애플리케이션에 요청을 보내도록 하여
- 피해자 대신 선택된 작동을 수행하도록 합니다.
- 특별한 공격 포인트가 없습니다.
- HTTP 트래픽 변조하지 않고, 문자나 인코딩 기법을 사용하지 않습니다.
- 브라우저에서 사용자 몰래 요청이 일어나게 강제하는 공격입니다.
